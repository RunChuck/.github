# 러닝한다이 (Running Handai)

부산의 다채로운 매력을 따라 달리는 데이터 기반 러닝코스 추천 플랫폼, [**러닝한다이**](https://runninghandai.com/)입니다

## 프로젝트 성과

- 🏆 2025 관광데이터 활용 공모전 **우수상** 수상
- 🎖️ 2025 관광데이터 활용 공모전 **부산관광공사 특별상** 수상
